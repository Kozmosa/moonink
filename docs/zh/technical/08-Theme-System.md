# 08 主题系统

## 1. V1 主题定位

MoonInk V1 内置一个官方主题。

## 2. 设计目标

- 完整到足以支持真实使用；
- 视觉上与 MoonInk 品牌语言一致；
- 内部上足够模块化，以支撑未来主题扩展。

## 3. 主题职责

官方主题应提供：

- 首页与落地页布局；
- 文档页模板；
- 文章页模板；
- 导航组件；
- 搜索界面；
- 排版与静态资源样式。

## 4. 品牌一致性

主题应遵循 `docs/Brand_Design.md` 中定义的色板与气质，尤其是克制、留白、带纸感的暖白背景，以及安静而精确的视觉表达。

## 5. 公共 API 立场

V1 主题系统应在内部被实现为可替换结构，但暂不宣告为稳定的公共扩展 API。
