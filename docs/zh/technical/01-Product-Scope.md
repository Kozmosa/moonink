# 01 产品范围

## 1. 定位

MoonInk 是一个基于 MoonBit 的静态站点生成器，第一版聚焦于面向个人作者的知识型内容发布。

## 2. 第一版范围

### 包含内容

- Markdown 内容
- TOML 配置
- 双轨站点结构
- frontmatter 元数据
- 自动与手工结合的导航控制
- 内部链接校验
- WikiLink 支持
- 一个官方主题
- 本地静态搜索索引
- `new`、`build`、`serve` 命令

### 不包含内容

- 第三方插件
- 完整多语言支持
- 多主题
- 部署自动化
- 增量构建
- 可视化知识图谱

## 3. 产品边界原则

如果一项能力不能直接改善内容组织、构建可预测性或基本发布可用性，那么它不应进入 V1。

## 4. 用户价值

第一版应让用户能够：

- 将稳定的知识笔记组织成文档树；
- 并行发布文章型内容；
- 以更高信心使用内部链接；
- 对已发布内容进行本地搜索；
- 以较低的心智负担运行整套工具。
