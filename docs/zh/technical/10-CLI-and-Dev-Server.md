# 10 CLI 与开发服务器

## 1. V1 的 CLI 命令面

- `moonink new`
- `moonink build`
- `moonink serve`

## 2. 命令职责

### `new`

创建一个带默认配置、样例内容与官方主题资源的起始项目。

### `build`

运行完整静态生成流水线，并输出最终站点结果。

### `serve`

运行本地开发服务器以预览生成站点。V1 可以从简单的“重建并刷新”行为起步，而不是一开始就追求复杂热重载。

## 3. CLI 输出风格

CLI 信息应简洁、冷静且利于诊断。成功、警告与失败输出都必须帮助作者快速定位内容问题。

## 4. 演进路径

待核心命令面稳定后，未来可加入 `check`、`deploy`、`doctor` 以及主题或插件相关工具命令。
