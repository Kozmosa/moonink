# 05 路由与导航

## 1. 路由目标

- URL 可预测；
- 页面规范标识稳定；
- 输出结构清晰；
- 支持混合内容类型。

## 2. 路由策略

MoonInk V1 应优先为内容页面采用目录式输出 URL。这种方式可读性更好，也更符合文档站浏览习惯。

## 3. 导航策略

MoonInk V1 采用混合策略：

- 默认根据内容结构自动生成导航；
- 允许通过配置进行人工覆盖或精调。

## 4. 文档导航

文档页应以层级树方式呈现，默认基于文件夹结构生成，除非配置显式调整这棵树。

## 5. 文章导航

文章不应被强行塞进文档侧边树中。它们应出现在文章列表、标签列表或首页聚合区域。

## 6. 冲突规则

路由系统必须检测：

- 重复的输出路径；
- 重复的页面标识；
- 指向不存在页面的导航条目。
